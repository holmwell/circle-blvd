Use mileposts to keep track of deadlines and special occasions, or add some structure to your long list. Want to get something done before a particular date? Make a milepost and put the task above it.

Try using mileposts to keep track of tasks you want to maybe do later. Mileposts can be especially helpful when combined with the roadmap mindset, outlined in the planning section of the tour.

Archive mileposts by moving them above the black task and using the icon.

Do you have a set of tasks that repeat a lot? Save this as a checklist!

Use the checklists page to create (or edit) them. Add them to your main list by using the standard entry panel (and selecting the checklist option).

Adding many tasks

If you just want to enter a lot of tasks and mileposts at the same time, take a look at the "many" option on the entry panel.

Tasks that are owned by you are marked by a blue, dotted edge.

Archive completed tasks

When a task is done, and you want it off the list, using the icon (next to the task summary) will move it into the archives. Once a task is in the archives, it cannot be changed.

Comment on tasks

Commenting on a task will email all the people involved: The person who created the task, the task owner, and the people who have previously commented on this particular task.

Quick, back-and-forth converations aren't really a focus of Circle Blvd, since there are so many other websites or apps (or texting) for that sort of thing these days. A search for "group chat" on your favorite search engine will show a few options that might compliment your use of Circle Blvd.

We suggest using comments to ask the task creator clarifying questions, or to post decisions that have been made about a task.

Notify task owners

You may have noticed the question mark next to the task owner's name. At Circle Blvd, just because you typed someone's name into a box, that doesn't mean a task is going to be done by that person.

If you want, use the icon to send a short "please look at this" email to the potential owner. This icon was created, as a courtesy, for those times when someone asks you to remind him or her to do something.

We suggest using this icon while you're having a conversation with the teammate. People can decide what email address they want to use to receive these notifications.
